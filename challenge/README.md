# Fix My Code Challenge

## Description

Fix My Code est un projet dont l'objectif est d'analyser, comprendre et corriger du code existant contenant des erreurs.

Contrairement aux autres projets où il faut développer une solution complète, ici le travail consiste à identifier les bugs présents dans un programme et à les corriger en modifiant le moins de code possible.

Ce projet permet de développer des compétences essentielles en :

* Lecture de code existant
* Débogage (debugging)
* Analyse de la logique d'un programme
* Correction de bugs
* Validation et tests

## Requirements

* Ubuntu 20.04 LTS
* Tous les fichiers doivent se terminer par une nouvelle ligne
* Utilisation des éditeurs autorisés :

  * vi
  * vim
  * emacs

## Tasks

### 0. FizzBuzz

Corriger l'implémentation Python du programme FizzBuzz.

#### Expected behavior

* Afficher `Fizz` pour les multiples de 3
* Afficher `Buzz` pour les multiples de 5
* Afficher `FizzBuzz` pour les multiples de 3 et de 5

#### Example

```bash
$ ./0-fizzbuzz.py 15
1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz 11 Fizz 13 14 FizzBuzz
```

## Learning Objectives

À la fin de ce projet, je suis capable de :

* Lire et comprendre un code écrit par un autre développeur
* Identifier rapidement les erreurs de logique
* Corriger un bug sans réécrire l'application
* Vérifier qu'une correction ne casse pas les autres fonctionnalités
* Utiliser une démarche de débogage efficace
